# 变更：新增扩展认证接入

## Why
扩展需要复用 Web 认证体系完成登录/注册/登出，并在 prod/dev/local 环境下读取会话状态。

## What Changes
- 为 `/api/auth` 与 `/api/rpc` 增加可控的跨域允许列表（支持携带凭证）。
- 新增 Web 登出页面供扩展触发。
- 补充环境变量与域名配置说明。
- 明确扩展侧使用的会话端点 `/api/rpc/me`。
